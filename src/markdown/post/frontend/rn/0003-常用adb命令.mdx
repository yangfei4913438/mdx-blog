### 一、启动

```bash
adb start-server
```

### 二、停止

```bash
adb kill-server
```

### 三、查看连接到这个系统的设备

```bash
adb devices
```

### 四、端口映射（把手机端口映射到电脑上）

> 如果已经使用虚拟设备，可能出现端口被占用的情况，关掉虚拟设备，重新执行一下端口映射即可。

```bash
# 手机的8081 映射到电脑上
adb reverse tcp:8081 tcp:8081
```

### 五、进入沙盒（进入已经连接的安卓设备）

```bash
$ adb shell
 nuwa:/ $
```

  - 推送文件到手机中

    ```bash
    adb push /xx.jpg /sdcard/
    ```

  - 从手机往外取文件

    ```bash
    abd pull /sdcard/xx.jpg ./
    ```
