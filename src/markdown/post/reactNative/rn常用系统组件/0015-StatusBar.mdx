> `StatusBar`组件，用于控制状态栏。

### 属性

- barStyle 状态栏的文字颜色
- backgroundColor 状态栏的背景色
- animated 是否开启状态栏动画
- translucent 是否穿透状态栏下方（需要状态栏背景色改成带透明度的）
- hidden 是否需要隐藏状态栏
- api
  > 这里直接写，不需要通过ref来设置
  ```javascript
    StatusBar.setBackgroundColor('white');
    StatusBar.setBarStyle('dark-content');
    StatusBar.setHidden(false);
    StatusBar.setTranslucent(false);
  ```

### 例子

```javascript
import { StatusBar, Text, View } from 'react-native';
import { useEffect } from 'react';

const StatusBarDemo = () => {
  useEffect(() => {
    setTimeout(() => {
      // 直接修改状态栏的可见状态
      StatusBar.setHidden(true);
    }, 1000);

    setTimeout(() => {
      StatusBar.setHidden(false);
    }, 3000);

    setTimeout(() => {
      StatusBar.setHidden(true);
    }, 5000);
  }, []);

  return (
    <View className='w-full h-full'>
      <StatusBar
        barStyle='light-content' // 背景类型: 白色 light-content 灰色 dark-content
        backgroundColor='#00000060' // 状态栏的背景色，可以设置透明（translucent为 true 的时候设置透明）
        animated={true} // 是否开启状态栏动画
        translucent={false} // 状态栏，安卓一般是单独占位，ios 一般是浮动在上方。这里可以统一设置
        hidden={false} // 是否隐藏状态栏
      />
      <View className='w-full h-full justify-center items-center bg-[#ff990080]'>
        <Text className='text-3xl scale-150'>占位数据</Text>
      </View>
    </View>
  );
};

export default StatusBarDemo;
```
